# MeatAndGreet

## Setup

1. Clone the repository.
2. Create a new file in the root directory, called "config.js".
3. Insert your OpenAI API Key as follows:

"""
// for environment variables
// not pushed to git

const config = {
    OPENAI_API_KEY: 'YOUR-API-KEY-HERE',
  };
  
export default config;
"""

4. run npm install
5. run npm start
